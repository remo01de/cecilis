XSS-Schutz Test für Cecilia Chat

Über diesen Test

Diese Seite testet den XSS-Schutz (Cross-Site Scripting) im Cecilia Chat-System.

XSS-Angriffe versuchen, bösartigen JavaScript-Code in Webseiten einzuschleusen.

Status: ✓ XSS-Schutz aktiviert

Test-Szenarien

1. Script-Tag Injection

<script>alert('XSS')</script>

2. IMG mit JavaScript

<img src=x onerror="alert('XSS')">

3. HTML-Injection

<h1>Injected Heading</h1>

4. Event-Handler

<div onclick="alert('XSS')">Click me</div>

5. JavaScript-URL

<a href="javascript:alert('XSS')">Click</a>

6. Normale Nachricht (erlaubt)

Hallo Cecilia! Wie geht es dir? ✨
